Creating Wonder-less. A chorme extension to limit opened tabs.

Wonder-less is a Chrome extension that limits the tabs opened. 
I noticed that once I get past a certain point when opening tabs, I become distracted, overwhelmed, and less focused.

Wonder-less is the solution.

Now the user can have multiple options to increase the flexibility and not limit completely the user, avoiding getting frustrated with the app. While not completely limiting the user, the Chrome extension should always keep the user mindful of the number of tabs opened and show alerts once that number is reached.

 

I have kept the maximum number of tabs that can be opened to 20. Because giving the user flexibility on that aspect defies the extension's purpose.
If for some reason, the user wants to go over that limit, they can deactivate the extension.


This project did not progress beyond the design stage due to the release of Chrome's V3 update, which restricts access to certain features, including the ability to automatically close tabs.

But as I said in the Maker's Mindset blog: 

If you fail, that counts as done.
